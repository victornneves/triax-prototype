**Change 1 — Add textarea ref and helper functions in ProtocolTriage.jsx:**

```javascript
const textareaRef = useRef(null);
```

Add two new functions before the JSX return (near the other handler functions):
```javascript
const handleTextareaInput = () => {
    const el = textareaRef.current;
    if (!el) return;
    el.style.height = 'auto';
    el.style.height = el.scrollHeight + 'px';
};

const handleTextareaKeyDown = (e) => {
    if (e.key === 'Enter' && !e.shiftKey) {
        e.preventDefault();
        handleSendMessage();
    }
    // Shift+Enter: browser default inserts newline — no action needed
};
```

Add a useEffect to reset textarea height when inputText is cleared (handles both Enter-send and button-send paths, per Pitfall 3 in RESEARCH.md):
```javascript
useEffect(() => {
    if (!inputText && textareaRef.current) {
        textareaRef.current.style.height = 'auto';
    }
}, [inputText]);
```

**Change 2 — Replace input element with textarea in JSX (lines 919-927):**

Replace:
```jsx
<input
    type="text"
    value={inputText}
    onChange={(e) => setInputText(e.target.value)}
    onKeyDown={(e) => e.key === 'Enter' && handleSendMessage()}
    placeholder="Digite a queixa do paciente..."
    disabled={loading}
    className="chat-text-input"

```

With:
```jsx
<textarea
    ref={textareaRef}
    value={inputText}

    placeholder="Digite a queixa do paciente..."
    disabled={loading}
    className="chat-text-input"

/>
```
