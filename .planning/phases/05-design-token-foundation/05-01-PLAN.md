**2. Add FOUC prevention script to `index.html`** — inside `<head>`, BEFORE any CSS/script tags:
```html
<script>
  (function() {
    try {
      var t = localStorage.getItem('triax-theme') || 'light';
      document.documentElement.setAttribute('data-app-theme', t);
    } catch(e) {
      document.documentElement.setAttribute('data-app-theme', 'light');
    }
  })();
</script>
```
Place it after `<title>Triax</title>` and before `</head>`. Keep all existing elements (`meta charset`, `link rel="icon"`, `meta viewport`, `title`).
