```jsx
const getRoleBadgeClass = (role) => {
    const r = role?.toLowerCase();
    if (r === 'admin') return 'role-badge--admin';
    if (r === 'tenant_admin') return 'role-badge--tenant-admin';
    if (r === 'doctor' || r === 'medico') return 'role-badge--doctor';
    if (r === 'nurse' || r === 'enfermeiro') return 'role-badge--nurse';
    return 'role-badge--default';
};
```
Note: The current Header.jsx does not actually render a role badge visually (getRoleBadgeColor is defined but the badge rendering is not in the current code). If the function is defined but not called, simply delete it and the getRoleBadgeClass helper.
