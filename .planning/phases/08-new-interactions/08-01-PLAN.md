  1. Try `localStorage.getItem('triax-theme')` -- if truthy, use it
  2. Else check `window.matchMedia('(prefers-color-scheme: dark)').matches` -- if true, return `'dark'`

**2. Update FOUC script in `index.html`** (lines 8-16):
Replace the existing script block with:
```html
<script>
  (function() {
    try {
      var stored = localStorage.getItem('triax-theme');
      var t = stored
        ? stored
        : (window.matchMedia('(prefers-color-scheme: dark)').matches ? 'dark' : 'light');
      document.documentElement.setAttribute('data-app-theme', t);
    } catch(e) {
      document.documentElement.setAttribute('data-app-theme', 'light');
    }
  })();
</script>
```
Key change: when no localStorage value exists, check `prefers-color-scheme` media query before defaulting to light (per D-05).
